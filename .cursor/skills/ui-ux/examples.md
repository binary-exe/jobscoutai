# UI/UX Examples

## Empty state (jobs)

Checklist:

- icon or visual cue
- one-line title
- one-line guidance

## Error state (recoverable)

Checklist:

- plain language message
- “Try again” or “Back to …” action
- don’t leak stack traces to users

## Loading state

Checklist:

- skeleton blocks with `animate-pulse`
- keep layout stable (avoid jumps)
